# Tester quickstart (1 page)

## Install

- **iOS**: Open the TestFlight email → **View in TestFlight** → Install **Hit-A-Lick**.

## First launch

1. Allow notifications if prompted (optional).
2. Sign in with the account you were given (or create one if the build allows).

## What to try (5–10 minutes)

1. **Home / main feed** — Confirm games or props appear (not permanently empty).
2. **Elite Desk** (or props board) — Scroll, change sport if there is a filter, **pull down to refresh**.
3. **Settings / profile** — Sign out and sign back in once (session stability).

## Report issues

Include:

- **Device + iOS version** (Settings → General → About).
- **What you tapped** and **what happened** (crash, spinner forever, error text).
- **Screenshot** if something looks wrong.

**Do not** share passwords or payment info in feedback.

## Privacy

The app is an **analytics companion**; it does not place bets. See in-app terms / privacy links if present.
